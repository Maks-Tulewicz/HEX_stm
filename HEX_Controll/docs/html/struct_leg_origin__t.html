Detailed Description

Struktura pozycji origin pojedynczej nogi.

Każda noga ma inną pozycję punktu obrotu biodra (origin) względem centrum robota. Dodatkowo prawe nogi mają odwrócone kierunki obrotu.

Pochodzenie danych:

  • Wyeksportowane z URDF (Unified Robot Description Format)
  • Przeskalowane z metrów na centymetry (*100)
  • Potwierdzone w symulacji ROS
