enum WaveDirection_t

Kierunki ruchu w wave gait.

Enum definiujący dostępne kierunki poruszania się robota w trybie wave. Wave gait nie implementuje obrotów w miejscu - dla precyzyjnych obrotów zaleca się kombinację LEFT/RIGHT z krótkimi krokami.

Enumerator
WAVE_FORWARD 

Ruch do przodu (Y -= step_length)

WAVE_BACKWARD 

Ruch do tyłu (Y += step_length)

WAVE_LEFT 

Ruch w lewo - sideways (X += step_length)

WAVE_RIGHT 

Ruch w prawo - sideways (X -= step_length)
