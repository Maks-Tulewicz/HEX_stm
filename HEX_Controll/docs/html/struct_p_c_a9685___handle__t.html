Detailed Description

Struktura handlera PCA9685.

Przechowuje wszystkie niezbędne informacje o kontrolerze PCA9685. Każdy kontroler musi mieć własną instancję tej struktury.

Note
Struktura musi być zainicjalizowana przed użyciem funkcji PCA9685_Init()
Examples
pca9685_basic_example.c, pca9685_dual_controller.c, and tripod_basic.c.
